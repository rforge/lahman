Schools {Lahman}R Documentation

Schools table

Description

Information on schools players attended, by school

Usage

data(Schools)

Format

A data frame with 749 observations on the following 5 variables.

schoolID

school ID code

schoolName

school name

schoolCity

city where school is located

schoolState

state where school's city is located

schoolNick

nickname for school's baseball team

Source

Lahman, S. (2010) Lahman's Baseball Database, 1871-2012, 2012 version, http://baseball1.com/statistics/
